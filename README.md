## üö¶ Getting Started

### Prerequisites
```bash
# Android Studio 4.0+

# Android SDK 19 (KitKat)
```

### Installation
```bash
# Clone the repository
git clone https://github.com/yourusername/google-glass-revival.git

# Navigate to project
cd google-glass-revival

# Build Glass application
./gradlew assembleDebug

# Install on Glass device
```
